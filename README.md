# MPMM

### Dependency:

python == 3.7

pytorch == 1.9.1

fastnlp == 0.7.0

numpy == 1.21.2

pandas == 1.3.5

nltk == 3.7

seaborn == 1.7.3

### Dataset:

We use use the dataset in https://anonymousaaai2019.github.io/.

Please place the dataset you downloaded under the './data/stackoverflow' directory

### Usage:

`python train.py`
